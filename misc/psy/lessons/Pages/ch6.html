Memory

Memory is an active system that receives information from the senses, puts that information into a usable form, and organizes it as it stores it away, and then retrieves the information from storage.

Find out more

Encoding

Encoding Memories

The set of mental operations that people perform on sensory information to convert that information into a form that is usable in the brain’s storage systems.

Storing

Storing Memories

Operations people perform for holding onto information for some period of time.

Retrieval

Retrieval of Memories

Getting information that is in storage into a form that can be used at the moment.

Long Term Memories

Long term memories is where our system places more - or - less permanent information. The long term memory appears to be unlimited for all practical purposes.

Models of Memories

Determining how memory works since 1972

Information Processing Model

This model shows how information is handled through 3 different systems of memory: Encoding, Storing, and Retreiving

Parallel Distributed Processing (PDP) Model

In this model, information is processed at the same time in a large connection of neural networks.

Levels - of - Processing Model  

A Model of memory that assumes information that is deeply processed or processed according to its meaning will be remembered for a longer time.
